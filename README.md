# STM32F429I-DISC1-RACINGGAME

<img src="Images\RacingGame.jpg">

## Components list

- stm32f429i-disc1
- 2-Axis Joystick

## Usage

- Use STM32CubeIDE to import the project
- IO:
  - PC3 : X
  - PA5 : Y
## Gameplay
- Press START to begin the game
- Using the joystick to move the car around
- You have three lives
